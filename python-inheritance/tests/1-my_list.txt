Using ``MyList``
----------------

Import the class:

    >>> MyList = __import__('1-my_list').MyList

Test that MyList is a subclass of list:

    >>> issubclass(MyList, list)
    True

Create an instance:

    >>> my_list = MyList()

    <class '1-my_list.MyList'>

Test with empty list:

    >>> my_list = MyList()
